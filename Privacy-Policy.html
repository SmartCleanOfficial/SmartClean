Smart Clean Privacy Policy

Last updated: March 2026

Overview

Smart Clean is a Google Sheets add-on designed to help users quickly clean and standardize messy spreadsheet data. It provides tools to remove unnecessary spaces and standardize text formatting.

Information Collection

Smart Clean does not collect, store, transmit, or share any personal or sensitive user data.

The add-on only processes the data already present in the user’s Google Sheets and performs cleaning actions directly within the spreadsheet.

Data Usage

Smart Clean only uses spreadsheet data for the following purposes:

  • Removing leading and trailing spaces
  • Removing multiple consecutive spaces
  • Standardizing text capitalization
  • Standardizing email formatting (converting to lowercase)

All processing occurs within the user's Google Sheets environment.

Data Storage

Smart Clean does not store any user data on external servers or databases.

No spreadsheet data is saved, copied, or transmitted outside of Google Sheets.

Data Sharing

Smart Clean does not share any user data with third parties.

Permissions

The add-on may request permission to access Google Sheets in order to perform cleaning operations on the user's spreadsheet data.

These permissions are used only to perform the requested functionality.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any updates will be reflected on this page.
